Reuses the existing chunker + `Embedder`; **no migration** (`ContentChunk.kind` already supported `media_transcript`). Idempotent — media chunks ride the version's existing delete-then-rebuild, so re-ingest never duplicates. Graceful — an asset without a transcript (or a malformed `media_ref`) produces no media chunks and never fails ingestion. Tenant/version scoping unchanged (`TenantScoped` + RLS).
