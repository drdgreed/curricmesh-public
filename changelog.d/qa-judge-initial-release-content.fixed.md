- **QA judge now reads authored-course content too** — the initial-release path (Course Builder → publish) stores generated bodies in immutable `ContentVersion` rows with `change_set = None`, so the previous fix (which read `change_set`) still left the judge scoring blind on authored courses. `score_qa` now follows the `impact.initial_release.candidate_version_id` pointer and loads those bodies from the candidate version, rendered through the same bounded formatter. This is the path that actually produced the low QA scores on generated courses.
